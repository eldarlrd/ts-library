> - **[Mithril](https://mithril.js.org)**

>

## Install
```sh
yarn
```
## Develop
### Run Env.
```sh
yarn dev
```
### Prod. Build
```sh
yarn build
```
### Preview Prod. Build
```sh
yarn preview
```
### Format Code (ESLint + Prettier)
```sh
yarn format
```
## Deploy
```sh
yarn deploy
```
